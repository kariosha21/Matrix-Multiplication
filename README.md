# Matrix Operations in C

A menu-driven C program that performs fundamental matrix operations using functions and 2D arrays. This project demonstrates modular programming, matrix manipulation, and basic computational techniques in C.

## Features

* Matrix Addition
* Matrix Multiplication
* Matrix Transpose
* Function-based modular design
* 2D Array implementation
* User-friendly menu-driven interface

## Technologies Used

* C Programming Language
* Standard Input/Output Library (`stdio.h`)

## Concepts Covered

* Functions
* Arrays and 2D Arrays
* Nested Loops
* Matrix Computations
* Modular Programming
* User Input and Output

## How to Run

1. Compile the program:

```bash
gcc matrix_operations.c -o matrix
```

2. Run the executable:

```bash
./matrix
```

## Sample Operations

### Matrix Addition

Adds two matrices of the same dimensions.

### Matrix Multiplication

Multiplies two matrices when the number of columns in the first matrix equals the number of rows in the second matrix.

### Matrix Transpose

Converts rows into columns and columns into rows.

This project helps in understanding:

* Matrix representation using 2D arrays
* Function-based code organization
* Matrix arithmetic algorithms
* Efficient use of nested loops in C
